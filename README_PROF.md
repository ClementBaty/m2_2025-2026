# Description prof

## Barème : 100 points

### 60 points : travail individuel

- Contributions Git : 10 Points
- Clareté des messages de commit : 10 points
- Respect des PEP 8 et 257 : 10 Points
- Qualité du code : 10 points
- Utilisation frugale de l’IA : 20 Points

#### Pour l'utilisation frugale de l'IA :

- Nombre de tokens utilisés : 5 points
- Qualité des prompts : 5 points
- Nombre de demandes à l’IA : 5 points
- Effort de compréhension : 5 points

### 40 points : travail de groupe
- Respect du workflow Git : 10 points
- Respect du cahier des charges : 10 points
- Collaboration au sein du groupe : 10 points
- Intégration avec les autres groupes : 10 points

Note sur 20 au final
